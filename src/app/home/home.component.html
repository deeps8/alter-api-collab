APIs at ease.

About

The Application is based on API development .
It provides the user a platform where they can build, test, document and collaborate with the team.

Features

Simulating APIs

Automate manual tests and Communicate the expected behavior of an API by simulating endpoints. Users can test API.

Saving Requests

Users can make their own account and have personalized session pacific to a user only.
Users can save API endpoints with their respective response requests.

Creating Worspace

Logged in users can create different workspace for different projects and can invite the team members having the collaborative session.

Documenting APIs

Users have power to create documentation out of their saved endpoints, reponses and many more things.

Collborating

Users can invite the team members having the collaborative session.
Users can share among other members and download the APIs for future use.

Technology Used

Modules

  • User

    Users is a module where they have accessibility to the whole application and its features after logging or signing up to the app. Users have functionalities to create API requests by adding its endpoint, headers and body to it. In result it will show the response from that particular API , this helps in simulating the APIs locally.

  • Collaborative Workspace

    Workspace Module , it is the most important module in terms of having a team where everyone can collaborate together and give inputs in the API development lifecycle.
    A user having a workspace can invite other team members and can communicate with each other during the process.

  • Documenting

    The Documenting module is a very useful module. Everyone does not have knowledge of everything, developing API is not for everyone but we got you.
    This module provides a feature for documenting your saved endpoints with requests and responses which can be shared among your peers and can be downloaded for future use.
